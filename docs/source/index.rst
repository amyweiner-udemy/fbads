fbads
=====
.. image:: https://travis-ci.org/rpedigoni/fbads.png

\

Python client for the Facebook Ads API. **Under development!**


.. note::
    The Ads API is a restricted platform which may only be accessed by whitelisted Apps. `You can learn more about the program and apply online`_. You need to apply separately for each app.

.. _`You can learn more about the program and apply online`: https://developers.facebook.com/preferredmarketingdevelopers/
